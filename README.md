# Banking System (Console Java Project)

This is a simple console-based banking system made for a college project. Made by - Simran Kaur Tiwana(2510987041), Arpita Chaudhary(2510987007), Malika Gauba(2510987024), Moksh Jassal(2510987054)

## How to Run
Compile:
javac src/BankApp.java models/*.java

Run:
java -cp src BankApp

## Project Files
- BankApp.java → Main program
- models/ → Account, User, Transaction classes
- data/ → CSV files for users, accounts, transactions
